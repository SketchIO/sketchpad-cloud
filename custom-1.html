Custom Example 1

An example of a customized version of Sketchpad with a larger toolbar, specific clipart, and a set default theme.

The Sketchpad instance at the right of this page is implemented using this method. You get access to all the Sketchpad drawing tools, the file system, share pane, and user guide. This implementation is offered for free and includes a watermark to the Sketchpad interface which can be removed by upgrading your license.
